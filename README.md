# Amayeşir Language Analysis

This project presents an AI-assisted analysis of an agglutinative constructed language (Amayeşir).

## Overview
The study explores how artificial intelligence can be used to analyze and reconstruct meaning in a fictional language without parallel data.

## Methods
- Word frequency analysis
- Bigram / trigram modeling
- Conditional probability
- Morphological decomposition
- Semantic consistency modeling

## Results
- 422 unique words analyzed
- High-consistency semantic reconstruction achieved
- Fully unsupervised approach
